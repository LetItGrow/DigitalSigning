About this site

In this site you can sign a plan text document using a RSA private key locally with Javascript, using the jsrsasign library.

You can also generate RSA key pairs, and upload the public key to the server, asociated with a dummy username, to let you verify signing.

There is a test section where you can run some tests over signing and verification, and see the results.

The source code for this site can be seen in GitHub.

This site is made and design for testing purposes, it is not polished in any way. Don't use the information provided here for real secure applications.
